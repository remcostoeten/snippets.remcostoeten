## Python Example

<CodeBlock
  code={`def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Generate first 10 fibonacci numbers
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")`}
  language="python"
  fileName="fibonacci.py"
  badges={[{ text: 'Algorithm', variant: 'danger' }]}
  maxHeight="300px"
/>
