## Map super (win keyy) +  1,2,3,4,5,6 for workspace 1-6.

```bashrc copy
# Set Super + 1 to switch to workspace 1
gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-1 "['<Super>1']"

# Set Super + 2 to switch to workspace 2
gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-2 "['<Super>2']"

# Set Super + 3 to switch to workspace 3
gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-3 "['<Super>3']"

# Set Super + 4 to switch to workspace 4
gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-4 "['<Super>4']"

# Set Super + 5 to switch to workspace 5
gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-5 "['<Super>5']"

# Set Super + 6 to switch to workspace 6
gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-6 "['<Super>6']"

```
